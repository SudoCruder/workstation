True or false? 
            1.5 + 2.3 !== 3.8
          
          true

JS uses inaccurate floating point number.

In order to fix this always round off your numbers.

          (1.5 + 2.3).toFixed(2) !== 3.8
          => true

.toFixed(2) is not enough because that returns a string, '3.80'.

          +(1.5 + 2.3).toFixed(2) !== 3.8
          => false

The + sign converts the string into a number.
